> Fetches a specified n8n workflow definition from n8n.io and extracts its properties into individual data fields.

## 📌 Overview

This workflow is designed to programmatically retrieve an n8n workflow definition from the n8n.io platform using a provided workflow ID. Upon successful retrieval, it processes the raw JSON data of the workflow.

The workflow then systematically extracts various key properties from the definition, such as the workflow's name, description, tags, nodes, connections, and settings. Each of these extracted properties is then made available as a distinct output field, facilitating further automation, analysis, or integration with other systems.

The primary purpose of this workflow is to enable the automated fetching and parsing of n8n workflow definitions. ## 📤 Outputs

The workflow outputs a comprehensive set of fields, each representing a specific property of the retrieved n8n workflow:

*   **workflowName**: The name of the workflow.
*   **workflowDescription**: A description of the workflow.
*   **workflowTags**: An array of tags associated with the workflow.
*   **workflowNodes**: An array containing the definitions of all nodes within the workflow.
*   **workflowConnections**: An array detailing the connections between nodes.
*   **workflowSettings**: An object containing the workflow's configuration settings.
*   **workflowActive**: A boolean indicating if the workflow is active.
*   **workflowVersionId**: The version ID of the workflow.
*   **workflowIsTemplate**: A boolean indicating if the workflow is a template.
*   **workflowIsPublic**: A boolean indicating if the workflow is publicly accessible.
*   **workflowCreatedAt**: The timestamp when the workflow was created.
*   **workflowUpdatedAt**: The timestamp of the last update to the workflow.
*   **workflowId**: The ID of the retrieved workflow.
*   **workflowType**: The type of the workflow.
*   **workflowMeta**: Metadata associated with the workflow.
*   **workflowUser**: Information about the user who created/owns the workflow.
*   **workflowTeam**: Information about the team associated with the workflow.
*   **workflowCustomData**: Any custom data stored with the workflow.
*   **workflowData**: The raw data object of the workflow.
*   **workflowJSON**: The complete raw JSON definition of the retrieved workflow.
