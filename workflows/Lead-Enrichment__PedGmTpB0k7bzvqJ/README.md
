# Lead Enrichment

> This workflow automates the process of capturing, enriching, scoring, and routing leads from a web form, integrating with various services for qualification and sales engagement.

---

## 📌 Overview

This n8n workflow provides an end-to-end solution for managing incoming leads from a "Contact Us" web form. Upon submission, the workflow first validates the provided email address. If the email is deemed invalid, the lead is logged as rejected in a Google Sheet.

For valid leads, the workflow proceeds to enrich the data using external services like Hunter.io for email verification and a custom HTTP request for company information. It then checks an Airtable CRM to determine if the lead is from an existing customer. All this enriched data is used to calculate a comprehensive lead score, which in turn assigns a priority (High, Medium, or Low) to the lead.

Based on the assigned priority, the workflow intelligently routes the lead. High-priority leads are automatically processed: a detailed PDF report is generated, a notification is sent to a sales executive via Slack, a meeting is scheduled in Google Calendar, and a confirmation email is sent to the customer.
